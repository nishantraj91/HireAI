# HireAI

HireAI Frontend is a modern React.js application that provides an intuitive and responsive recruitment platform for job seekers and recruiters.

## Features

### Candidate Features

* User Registration & Login
* Browse Jobs
* Apply for Jobs
* Upload Resume
* Track Applications
* Personalized Dashboard

### Recruiter Features

* Recruiter Login
* Post Jobs
* Manage Job Listings
* Review Applications
* Recruiter Dashboard

### User Experience

* Responsive Design
* Modern UI Components
* Protected Routes
* Interactive Forms
* Bootstrap-Based Interface

## Tech Stack

### Frontend

* React.js
* JavaScript
* Bootstrap 5
* React Router DOM
* Axios
* React Icons

## Project Structure

src

├── components

## Installation

### Clone Repository

git clone <repository-url>

### Install Dependencies

npm install

### Run Application

npm run dev

# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
